# BR-M06-01 — 自動派工 5 因子權重

## Rule

Dispatch ranking 5 因子權重順序：distance + skill + rating + load + fairness（建議距離 / 空檔 / 技能優先於工資）。

## Source

- docs/_source/01-workorder-erp.md §M06 派工排程（BR-M06-01 + Q041）

## Constraints

- 5 因子 schema 在 M18 config 維護
- weight 可調整但 sum=1.0
- top-1 推薦
