# BR-M01-02 — 先建 Case 再進報價

## Rule

凡可能報價、派工、退款或客訴的 inquiry，必須先建立 Case（Case 為一切上游 entity 的前置條件）。

## Source

- docs/_source/01-workorder-erp.md §M01 客戶入口（BR-M01-02）

## Constraints

- 任何 ProblemCard / Quote / WorkOrder / RMA 必須能反查到 Case
- 阻擋 Coding：否
- Owner：客服主管
