# BR-M01-01 — Channel source 必填

## Rule

每個 Case 必須保留 source channel（LINE、電話、web、brand、store、dealer、builder、referral 之一）；無 channel 不可建 Case。

## Source

- docs/_source/01-workorder-erp.md §M01 客戶入口（BR-M01-01）

## Constraints

- source ∈ {LINE, 電話, web, brand, store, dealer, builder, referral}
- 阻擋 Coding：是
- Owner：客服主管
