# BR-M18-05 — Config audit retention ≥ 7 年

## Rule

每筆 config change 必須記錄 who / when / what diff / why，retention ≥ 7 年（合規 + 商業會計法）。

## Source

- ADR-0067 §audit；商業會計法 §83

## Constraints

- append-only ledger
- retention ≥ 7 year
- 4 必填欄位
