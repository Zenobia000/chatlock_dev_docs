# BR-M06-02 — 派工通知 timing（acceptance SLA）

## Rule

Acceptance SLA：normal 15 分鐘（前期 10 分鐘為較嚴格選項，需 supervisor 確認）、urgent 5 分鐘；逾時自動重派。派工通知 P95 ≤ 30s，紅色 ≤ 15s。

## Source

- docs/_source/01-workorder-erp.md §M06 派工排程（BR-M06-03 + Q043）

## Constraints

- 通知延遲 P95 ≤ 30s / 紅色 ≤ 15s
- 逾時 → auto re-dispatch
