# BR-A01-01 — Debounce 合併視窗

## Rule

LINE 連續訊息在 debounce window 內合併為 merged turn（預設 2 秒）；overlapping 訊息按時序合併。

## Source

- docs/_source/02-ai-chatbot-sync.md §A M01 進線Debounce（BR-M01-01）

## Constraints

- 預設 window=2s（M18 config 可調）
- output: merged turn
