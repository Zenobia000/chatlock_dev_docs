# BR-M18-04 — Config 變更 staged rollout

## Rule

高風險 config 變更必須採 staged rollout，每段 observation window ≥ 10 min，rollback ≤ 1 min。

## Source

- ADR-0067 §staged_rollout（M18 runtime governance）

## Constraints

- rollout stage ≥ 2 段
- rollback budget ≤ 1 min
- observation ≥ 10 min
