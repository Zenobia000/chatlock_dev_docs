# BR-A11-02 — DB reconnect 機制

## Rule

DB connection 斷線 → service 自動 reconnect（exponential backoff，max 5 次）；reconnect 期間 health = degraded。

## Source

- docs/_source/02-ai-chatbot-sync.md §A M11 部署健康

## Constraints

- backoff: 1s/2s/4s/8s/16s
- max retries=5
