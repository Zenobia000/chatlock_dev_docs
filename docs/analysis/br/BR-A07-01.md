# BR-A07-01 — Handoff 7 條件

## Rule

Handoff 觸發 7 條件：urgent / angry customer / 高金額 / 保固不明 / refund / safety-legal / 3 次失敗循環。任一命中 → transfer_to_human。Handoff form 帶已知 facts（電話 / 地址 / 設備 / 照片）。

## Source

- docs/_source/02-ai-chatbot-sync.md §A M07（BR-M07-01）+ §M03（BR-M03-02）

## Constraints

- 7 條件 OR
- handoff form 必填 4 facts
