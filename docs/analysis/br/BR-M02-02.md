# BR-M02-02 — 保固用 Device record

## Rule

主鎖與高價零件必須建立 Device record，包含 brand、model、serial、purchase/install/warranty_date。

- docs/_source/01-workorder-erp.md §M02 客戶地址設備（BR-M02-02）

## Constraints

- warranty_date 為保固判定起點
- Owner：客服主管 / 品牌
