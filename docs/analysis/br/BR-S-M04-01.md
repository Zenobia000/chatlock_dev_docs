# BR-S-M04-01 — Confirmed PC → created WO

## Rule

ConvertToWO 接受 status=confirmed 的 ProblemCard → 建立 status=created 的 WorkOrder；status=draft / pending 的 PC 不可轉。

- docs/_source/02-ai-chatbot-sync.md §S M04 ConvertToWO（BR-M04-01）

- output: WO.status=created
