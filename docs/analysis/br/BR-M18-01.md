# BR-M18-01 — Master configuration owner

## Rule

Service items、price tables、status codes、SLA、templates、roles、regions 需要明確 owner 與 approval。任一 owner 變更須走 ChangeRequest，rollout 採 BR-M18-04 staged policy（5/50/100, 30 min observation, fast-track 15 min）。

## Source

- docs/_source/01-workorder-erp.md §M18 System Admin（BR-M18-01）
- value-decisions 2026-05-28（rollout policy alignment）

## Constraints

- 每項 config 必須 owner 不為 null
- Owner：System admin / 主管
- Owner approval 寫入 audit ledger（BR-M18-05）
