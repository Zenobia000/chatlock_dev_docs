# BR-M17-01 — Can view / edit / approve matrix

## Rule

每個角色必須拆成 can-view / can-edit / can-approve；除 audited admin 外，不接受 all access。Segregation of Duty：同一 user 不應在無 second approval 下同時 create、approve、reconcile refund。

## Source

- docs/_source/01-workorder-erp.md §M17 Auth Audit（BR-M17-01 + BR-M17-02）

## Constraints

- 3 binary flags 矩陣
- SoD 強制 second approval
