# BR-A03-02 — Tool registry allowed/forbidden

## Rule

Tool registry 三層：allowed_tools / forbidden_tools / audit_required；forbidden 直接 reject，audit_required 必須寫 audit log。

## Source

- docs/_source/02-ai-chatbot-sync.md §A M03 ReAct Agent + KB tool registry

## Constraints

- 3 list 互斥
- forbidden ⊆ NEVER call
- audit_required → reasoning trace 必寫
