# BR-M04-05 — 報價有效期限分類

## Rule

報價有效期依案件類型雙 default：
- **一般 quote = 14 天**
- **急件 quote = 3 天**（含鎖具開鎖、緊急修繕、客戶 SLA-soft 內案件）

期滿自動 expired，需重新報價（不可 silent 延長）。

## Source

- docs/_source/01-workorder-erp.md §M04 報價價格（Q032）
- value-decisions 2026-05-28（業主裁決）

## Rationale

台灣居家修繕業普遍 14-30 天；鎖具急件 3-7 天。雙 default 對齊產業實況，比舊 7 天單一值精確。subagent 提案 7 天會打到一般 quote 客戶（過短）也打到急件 ops（過長）。

## Constraints

- 一般 default = 14 day
- 急件 default = 3 day
- 案件類型判定依 priority flag（M04 quote 表）
- 雙 default 皆 configurable via M18（rollout 走 BR-M18-04）
- expired 必須在 expire 後 1 hr 內標記，且觸發 LINE 通知客戶
