# BR-M04-05 — 報價有效期限分類

## Rule

報價有效期限依案件類型設定：一般 3 天、標準安裝 7 天、品牌/建商 15-30 天。

- docs/_source/01-workorder-erp.md §M04 報價價格（Q032）

- 預設天數可在 M18 config 中調整 [VALUE_DECISION_NEEDED]
