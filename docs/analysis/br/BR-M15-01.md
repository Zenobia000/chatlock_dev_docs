# BR-M15-01 — Exception return path 9 種

## Rule

每個 exception 必須選一個 return path：continue / requote / reschedule / reassign / new_workorder / cancel / refund / RMA / dispute。

## Source

- docs/_source/01-workorder-erp.md §M15 異常核准（BR-M15-01 + Q053 + G026）

## Constraints

- 9 enum 唯一
- exception 不自動 cancel — 先選 return path
